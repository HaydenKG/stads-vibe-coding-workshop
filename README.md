# Objective
This repo shall serve as demo for the Stads student association vibe coding workshop I created with other members of the stads education team.
Participating students will be provided with the opportunity to explore vibe coding with the objective of building their own product website using HTML, CSS and JS as simple entry languages.
The demo website is a basic product website of an imaginary STADS smoothie called _stoothie_.

## Workshop plan:
- day 1: Introduction to the goal, topic and technical aspects. Help students setting up their working environment and let them come up with ideas for their own product website and how to get started
- day 2: demonstrate the migration of the HTML, CSS and JS to svelte or Vue (loaded via CDN to reduce setup overhead) and reserve some time for practicing and problem solving with us tutors
- day 3: presentation of the results, feedback and learnings
